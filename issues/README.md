## 報告方法

新しい問題を発見した場合は、以下の形式でissueファイルを作成してください：

```
issues/XXX_issue_title.md
```

テンプレート：
- 問題の概要
- 詳細
- 再現手順
- 期待される動作
- 影響度
- 優先度
- 関連ファイル
- 提案される解決策

## 優先度定義

- **高**: ゲームの核心機能やUI/UXに重大な影響
- **中**: ユーザビリティや利便性に影響
- **低**: 軽微な問題や改善提案
